# simulator — 現実世界の写像を目指す物理エンジン

現実空間の事象をデジタル空間に再現し、**検証して遊ぶ**ためのシミュレータ。
地球の物理法則 — 力学・流体・熱・電磁気・量子・統計 — を忠実に再現する物理エンジンを中核とし、
その環境の上に物・人・乗り物・生物が存在する世界を目指す。

## 現在の状態: 設計フェーズ

本リポジトリには現在、物理エンジンの**設計書一式**(45 文書、日本語)が含まれる。
実装(Rust + WebAssembly、ブラウザ 3D デモ)は設計レビュー承認後に
[ロードマップ](docs/22-roadmap/01-phases.md)のフェーズ計画に従って行う。

## 設計の要点

- **スケールの階梯と有効理論**: 量子から剛体まで、各スケールで最良の理論をソルバとして実装し、
  理論間の受け渡し(結合)を明示的に設計する。「物理の全域を押さえる」ことの唯一実現可能な形
  ([docs/00-foundation/02-scale-ladder.md](docs/00-foundation/02-scale-ladder.md))。
- **検証可能性**: 全ドメインに解析解テスト(許容誤差つき)と保存則チェックを定義。
  エネルギー台帳が数値誤差を常時可視化する。
- **決定論**: 固定タイムステップ・シード付き乱数・状態ハッシュにより、同条件なら
  ビット単位で同じ結果。リプレイ・共有・回帰テストの基盤。
- **実装直前レベルの詳細度**: 各文書は支配方程式の導出、離散化の具体式、Rust 型定義、
  出典つきパラメータ表、擬似コードを含む。
