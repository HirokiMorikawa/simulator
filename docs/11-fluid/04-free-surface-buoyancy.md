# 流体 04. 自由表面・浮力・表面張力

crate: `sim-fluid`。「物が浮く・沈む」を Phase 1 から提供する集中定数浮力と、
その上位互換となる解像モデル(SPH/格子)の関係を定義する。

## 1. 担う現実の現象

木片は浮き石は沈む、氷山の一角(9 割が水面下)、船の積載と喫水、水面の波紋、滴・泡。
遊び方の例: 密度を変えて喫水を予測→実測、氷が融けても水位が変わらないこと(アルキメデス)の確認。

## 2. 支配方程式

### 2.1 アルキメデスの浮力(導出)

静水圧 $p = p_0 + \rho_f g d$($d$: 深さ)を物体表面で積分すると、圧力の合力は

$$\mathbf{F}_b = -\oint p\,\hat{n}\,dA = -\rho_f \mathbf{g}\, V_{sub}$$

(ガウスの定理。$V_{sub}$: 水面下の体積)。**排除した流体の重さに等しい上向きの力**。
作用点は排除体積の重心(**浮心** center of buoyancy)— 重心とずれると復原/転覆モーメントを生む
(船の安定性 = メタセンタ高さ)。これを正しく出すため浮心の計算を必須とする。

### 2.2 浮沈の判定と平衡

平均密度 $\bar\rho < \rho_f$ で浮く。浮遊平衡の喫水は $\rho_f V_{sub} = m$ を満たす。
例: 一辺 $a$ の立方体(密度比 $r$)の喫水は $ra$。氷($\rho = 917$)は水面下 91.7%。

### 2.3 表面張力(Phase 4+)

Young-Laplace: 曲面圧力差 $\Delta p = \sigma_s \kappa$($\kappa$: 平均曲率)。
SPH では CSF(continuum surface force)モデル: 表面近傍粒子に
$\mathbf{f} = -\sigma_s \kappa \hat{n}\,\delta_s$ を付与。カラーフィールド勾配で $\hat{n}, \kappa$ を推定。

## 3. 状態表現 — 静的水域(Phase 1 の水)

流体を解像せず「水がある領域」だけを定義する軽量モデル:

```rust
pub struct StaticWaterRegion {
    pub aabb: Aabb,                 // 水域 (上面 y = water_level が自由表面)
    pub water_level: f64,
    pub density: f64,               // 998.2
    pub flow_velocity: Vec3,        // 一様流 (川) Phase 3
}
```

## 4. 数値解法 — 部分浸水体積と浮心

剛体形状ごとに「水平面より下の体積と重心」を解析計算する:

| 形状 | $V_{sub}$, 浮心 |
|---|---|
| 球 | 球冠(spherical cap): $V = \frac{\pi h_c^2}{3}(3r - h_c)$、浮心は軸上の解析式 |
| 箱(軸平行でない一般姿勢) | 平面による凸多面体切断(頂点分類 + 切断多角形で閉じる)→ 四面体分割で体積・重心 |
| カプセル・複合 | 部品ごとに計算して合成 |
| 凸メッシュ | 同じ切断アルゴリズム |

適用する力(すべて浮心・接触点に):

```text
buoyancy_step(body):
  (V_sub, c_buoy) = submerged_volume(shape, transform, water_level)
  if V_sub == 0: return
  apply force  F_b = −ρ_f * g * V_sub * ŷ    at c_buoy
  # 水中抗力・回転抗力 (集中定数、[05] の式を水の ρ,μ で)
  apply drag   F_d = −½ ρ_f C_d A(V_sub比で補間) |v_rel| v_rel   at c_buoy
  apply angular drag τ = −c_ω ρ_f ω |ω|      # 経験係数、§9
```

- 水面通過の不連続を避けるため、$V_{sub}$ は姿勢・位置の連続関数として計算する(切断の厳密計算が効く)。
- 波・流れ(Phase 3+)では `water_level` を場 $\eta(x,z,t)$ に一般化(単純な正弦波 or 浅水方程式は将来)。

### 4.1 解像モデルとの関係

SPH・格子ソルバが有効な水域では浮力は圧力積分から**創発**するため、本モデルは**適用しない**
(二重計上禁止、[20-integration/01](../20-integration/01-coupling-matrix.md))。
同一シーンを両モデルで実行し喫水を比較するデモを検証に使う(§7)。

## 5. 適用スケールと限界

- 静的水域モデルは「水は乱されない」近似: 物体が水を動かす反作用(波・水流)は無い。
  物体サイズ ≪ 水域で妥当。飛び込みのスプラッシュが見たければ SPH シーンを使う、と UI で案内。
- 動的な喫水変動(ヒーブ・ピッチの流体減衰)は集中定数抗力の精度(±30%)に依存。
- 表面張力モデルは mm スケール以下で定量精度を主張しない([03-sph.md](03-sph.md) §5)。

## 6. 他ドメインとの結合

- 剛体: 力・トルク(force_accum 経由)。
- 熱: 水温(StaticWaterRegion に温度を持たせ、浸水体への対流熱伝達
  [12-thermal/02](../12-thermal/02-heat-transfer.md) §4 の水中係数を適用)。氷が融けるデモへ。

## 7. 検証

- 立方体喫水: 密度比 0.5 → 喫水 50% ± 1%(静的モデル)/ ± 5%(SPH)。**両モデルの相互一致**を確認。
- 球冠体積: 解析式とのユニットテスト($10^{-12}$)。
- 復原: 重心を上げた箱が転覆する臨界高さがメタセンタ計算と一致(± 5%)。
- 減衰振動: 浮体の上下振動周期 $T = 2\pi\sqrt{m/(\rho_f g A_{wl})}$($A_{wl}$: 水線面積)± 5%、
  減衰は抗力係数依存(定性)。
- 氷融解で水位不変(熱結合の統合デモ、± 1 mm)。

## 8. 実装フェーズ対応

Phase 1: StaticWaterRegion + 球・箱の切断体積 + 浮力/水中抗力。Phase 3: 一様流・波高場。
Phase 4: SPH 浮力の創発と相互検証、表面張力。
