# 力学 04. 摩擦 — クーロンモデル・摩擦円錐・素材係数表

crate: `sim-mechanics`。[03-contact-solver.md](03-contact-solver.md) の接線方向の解法。

## 1. 担う現実の現象

置いた物が滑らない、押すと滑り出す、斜面の滑り出し角、ブレーキ、タイヤのグリップ、
氷の上のカーリングストーン。遊び方の例: 斜面の角度を上げて $\tan\theta = \mu_s$ で滑り出す瞬間を測る、
素材を変えて制動距離を比べる。

## 2. 支配方程式

**クーロン摩擦(アモントン=クーロンの法則)**:

- 静止摩擦: 接線力 $|\mathbf{F}_t| \le \mu_s N$ の範囲で相対滑り速度をゼロに保つ。
- 動摩擦: 滑り中は $\mathbf{F}_t = -\mu_k N \,\hat{\mathbf{v}}_t$($\hat{\mathbf{v}}_t$ は滑り方向)。
- $\mu$ は接触面積・速度に(一次近似で)依存しない。$N$ は垂直抗力。

インパルス定式化では、法線インパルス $j_n$ に対し接線インパルス $\mathbf{j}_t$ が**摩擦円錐**
$|\mathbf{j}_t| \le \mu j_n$ 内に制約される。

### 2.1 円錐の箱近似(採用)

円錐制約は非線形なので、接線基底 $(\hat{\mathbf{t}}_1, \hat{\mathbf{t}}_2)$
([01-math/01-linear-algebra.md](../01-math/01-linear-algebra.md) の `orthonormal_basis`)で
2 本の独立制約 $|j_{t1}| \le \mu j_n$, $|j_{t2}| \le \mu j_n$ に分解する(**箱近似**)。
誤差: 対角方向で最大 $\sqrt{2}$ 倍の摩擦を許す。日常シーンでは知覚されないが、
方向依存が問題になる場合(タイヤ)は円錐への射影(クランプ後にノルムで再スケール)を
オプション実装する(Phase 4 の車両で再評価)。

## 3. 状態表現

[03-contact-solver.md](03-contact-solver.md) §3 の `tangent_impulse: (f64, f64)` と `friction: f64`。
静止/動摩擦の区別は係数 2 本(§9)を持ち、ソルバでは接触点の滑り速度
$|\mathbf{v}_t| < v_{stick}$ なら $\mu_s$、それ以上は $\mu_k$ を使う(ヒステリシス付き切替)。

### 3.1 係数の結合則

異種材料 A-B の接触係数は**幾何平均** $\mu_{AB} = \sqrt{\mu_A \mu_B}$ を既定とする
(Box2D 等の慣例。物理的には材料ペア固有値が正しい)。
**ペア表オーバーライド**を MaterialDb に持ち、実測値が知られる組(ゴム-アスファルト等)は
ペア値を直接指定する。反発係数も同様(結合則は $\max$ でなく幾何平均を採用し、ペア表を優先)。

## 4. 数値解法(接線 solve の具体式)

法線 solve([03](03-contact-solver.md) §4.2)の直後に各接線方向で:

```text
solve_tangent(point p, dir k ∈ {t1, t2}):
  v_t = t_k · [(v_B + ω_B×r_B) − (v_A + ω_A×r_A)]
  Δj  = −v_t * p.tangent_mass[k]          # 目標: 相対接線速度 0 (= 静止摩擦を試みる)
  j_old = p.tangent_impulse[k]
  limit = μ * p.normal_impulse            # 同じ反復中の最新 j_n を使う
  p.tangent_impulse[k] = clamp(j_old + Δj, −limit, +limit)
  Δj = p.tangent_impulse[k] − j_old
  apply impulse Δj·t_k
```

- クランプに達しない → 静止摩擦(滑りゼロを達成)。達する → 動摩擦(円錐面上、滑りが残る)。
  静止/動の切替が**クランプから自然に出る**のがこの定式化の利点。
- 接線基底はステップ間でできるだけ連続に保つ(前ステップの $\hat{\mathbf{t}}_1$ を法線に直交化して再利用)。
  基底が毎ステップ回転すると蓄積インパルスの意味が変わり、warm starting が劣化するため。

### 4.1 転がり摩擦(Phase 2)

転がる球・円柱の減速。トルク制約 $|\tau_{roll}| \le \mu_{roll}\, N\, r$ として同じクランプ構造で実装。
$\mu_{roll}$ は $10^{-3}$〜$10^{-2}$ オーダー(鋼球/鋼板 ≈ 0.001、タイヤ/路面 ≈ 0.01–0.03。
出典: Marks' Standard Handbook)。これが無いと球が永遠に転がり続ける。

## 5. 適用スケールと限界

- クーロンモデルは乾摩擦の一次近似。速度依存(ストライベック効果)、温度依存、
  スティックスリップの動力学、接触面の真実接触面積の物理(凝着理論)は捨象する。
  必要になれば $\mu(v, T)$ への拡張点を MaterialDb に確保(温度依存は熱結合デモで使う可能性)。
- 潤滑(流体膜)は対象外(流体ドメインとの将来結合課題)。

## 6. 他ドメインとの結合 — 摩擦発熱

滑り接触の散逸仕事率 $\dot{W} = \mu_k N \, |\mathbf{v}_t|$。ステップあたりの発熱量は
インパルスから $\Delta Q = |\mathbf{j}_t \cdot \mathbf{v}_t^{post}|\,$(接線インパルス×滑り速度)で計上し、
接触イベント経由で熱ドメインに渡す(両体へ 1:1 か熱浸透率比 — [12-thermal/02](../12-thermal/02-heat-transfer.md) §6 で規定)。
ブレーキが熱くなる・こすると温まるデモの基盤。

## 7. 検証

- 斜面テスト: 傾斜 $\theta$、$\tan\theta < \mu_s$ で静止(速度 $<10^{-4}$ m/s)、
  $\tan\theta > \mu_s$ で滑り、加速度 $= g(\sin\theta - \mu_k\cos\theta) \pm 2\%$。
- 制動距離: 初速 $v_0$ の箱の停止距離 $= v_0^2/(2\mu_k g) \pm 2\%$。
- エネルギー: 摩擦仕事の合計が運動エネルギー減少と一致($<1\%$)— 熱結合の検算にもなる。
- 箱近似の等方性テスト: 45° 方向滑りで摩擦力の過大が $\le \sqrt{2}$ 倍に収まること(既知の近似誤差の確認)。

## 8. 実装フェーズ対応

Phase 1: 箱近似クーロン摩擦 + 幾何平均結合則。Phase 2: 転がり摩擦、ペア表、ヒステリシス切替。
Phase 4: 円錐射影(車両)、速度・温度依存 $\mu$ の検討。
