# 統計 03. 拡散・ブラウン運動 — ランジュバン方程式と揺動散逸定理

crate: `sim-statistical`。ミクロのゆらぎがマクロの拡散になる過程の両側を実装する。

## 1. 担う現実の現象

水中の花粉・微粒子のジグザグ運動(ブラウン運動)、インクの拡散、香りが部屋に広がる、
コロイドの沈降平衡。遊び方の例: 粒子径を変えてジグザグの激しさを見る、
平均二乗変位が時間に比例することを測る(アインシュタインの関係の再発見)、
インク滴の広がり半径 $\propto\sqrt{t}$。

## 2. 支配方程式

### 2.1 ランジュバン方程式(ミクロ側)

微小粒子(質量 $m$、半径 $r$)の運動方程式に、溶媒分子の衝突を
「平均(粘性抵抗)+ゆらぎ(ランダム力)」に分けて書く:

$$m\dot{\mathbf{v}} = -\gamma\mathbf{v} + \sqrt{2\gamma k_BT}\,\boldsymbol\xi(t) + \mathbf{F}_{ext}$$

$\gamma = 6\pi\mu r$(ストークス抵抗)、$\boldsymbol\xi$: 白色ノイズ($\langle\xi_i(t)\xi_j(t')\rangle = \delta_{ij}\delta(t-t')$)。

**揺動散逸定理**: ノイズ強度 $\sqrt{2\gamma k_BT}$ は抵抗係数と温度で**一意に決まる**
(平衡分布がボルツマン分布になる条件)。抵抗とゆらぎは同じ分子衝突の 2 つの顔 —
この必然性が本ドメインの中心メッセージ。

### 2.2 拡散方程式(マクロ側)

多数のブラウン粒子の密度 $c(\mathbf{x},t)$ は

$$\frac{\partial c}{\partial t} = D\nabla^2 c, \qquad D = \frac{k_BT}{\gamma} \;(\text{Stokes-Einstein})$$

平均二乗変位(3D): $\langle|\Delta\mathbf{x}|^2\rangle = 6Dt$(長時間)。
アインシュタインの 1905 年の関係式 — 分子の実在の歴史的証拠をデモで追体験する。

## 3. 状態表現

- ブラウン粒子 = 通常の剛体(微小)+ `BrownianForce` コンポーネント、
  または軽量 `ParticleSet`(多数の場合)。
- インク拡散 = `Grid3<f64>` の濃度場(流体の移流と重ねられる: 移流拡散方程式)。

## 4. 数値解法

### 4.1 ランジュバン積分 — BAOAB スキーム

過減衰でない一般形に対する高精度分割法(Leimkuhler-Matthews):

```text
B: v += (F/m)(dt/2)
A: x += v(dt/2)
O: v = c1*v + c2*√(kBT/m)*N(0,1)³   # c1 = e^{−γdt/m}, c2 = √(1−c1²)  (OU過程の厳密解)
A: x += v(dt/2)
B: v += (F/m)(dt/2)
```

- O 段が Ornstein-Uhlenbeck 過程の**厳密解**なので、大きな $\gamma\Delta t$ でも
  平衡分布(マクスウェル分布)を正確にサンプルする。
- 過減衰極限(水中微粒子: 慣性時間 $m/\gamma \sim 10^{-7}$ s ≪ $\Delta t$)では
  位置ランジュバン(Euler-Maruyama): $\Delta\mathbf{x} = \frac{\mathbf{F}}{\gamma}\Delta t + \sqrt{2D\Delta t}\,\mathbf{N}(0,1)$。
  切替は $\gamma\Delta t/m > 10$ で自動。
- 乱数は STREAM_STATISTICAL([01-math/04](../01-math/04-random.md))、粒子ごとにサブストリーム。

### 4.2 濃度場の拡散

陰的 Euler + PCG(熱伝導と同一コード、[12-thermal/02](../12-thermal/02-heat-transfer.md) §4.3)。
流体があれば semi-Lagrangian 移流と分割合成(移流拡散)。

## 5. 適用スケールと限界

- ランジュバンが有効: 粒子 ≫ 溶媒分子(nm 以上)かつブラウン力が他の力と競合する
  スケール(μm 以下)。判定式([01](01-micro-macro-bridge.md) §4)により、
  mm 以上の剛体には自動で付与しない(付けても効果ゼロだが計算の無駄)。
- 流体力学的相互作用(粒子間の溶媒媒介相互作用)は捨象(希薄コロイドのみ)。
- 回転ブラウン運動は Phase 5+(必要になれば同じ枠組みで)。

## 6. 他ドメインとの結合

- 力学: BrownianForce を force generator として注入(重力・静電力と重ねられる —
  沈降平衡 $c(h) \propto e^{-mgh/k_BT}$ のペラン実験デモ)。
- 熱: 温度 $T$ を熱ドメインから参照(温めるとブラウン運動が激しくなる)。
- 流体: 粘性 $\mu$ を媒質から参照。移流拡散。

## 7. 検証

- MSD: $\langle|\Delta x|^2\rangle = 6Dt$ の線形性と係数(± 3%、$10^3$ 粒子 × $10^4$ step)。
- 平衡分布: 調和トラップ中の位置分散 $= k_BT/k_{trap}$(± 2%)— 揺動散逸の直接検証。
- 沈降平衡: 高度分布の指数則(ペラン実験、± 5%)。
- 速度分布: マクスウェル分布への収束($\chi^2$ 適合)。
- 濃度場: ガウス解 $c(x,t) = \frac{M}{(4\pi Dt)^{3/2}}e^{-x^2/(4Dt)}$ との比較(< 2%)。
- 粒子系と場の整合: 同一 D で粒子ヒストグラム = 場の解(± $1/\sqrt N$)。
