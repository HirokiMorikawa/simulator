# 統計 01. ミクロ⇔マクロの橋 — アンサンブル・粗視化の方法論

crate: `sim-statistical`。統計力学のエンジン内での役割の総論。

## 1. 担う現実の現象

温度・圧力・エントロピーという「マクロ量の意味」。なぜ気体は容器に均一に広がるのか、
なぜ熱は高温から低温に流れるのか(確率の言葉での説明)、ゆらぎ(ブラウン運動・熱雑音)。
遊び方の例: 分子デモの圧力計の読みが $pV=Nk_BT$ に載る、2 区画の気体が混ざって戻らない
(エントロピー増大を状態数で見る)。

## 2. 支配方程式・原理

- **ボルツマン分布**: 温度 $T$ の平衡で状態 $i$ の確率 $P_i \propto e^{-E_i/(k_BT)}$。
  すべての有効モデル(アレニウス・マクスウェル分布・ダイオード式)の共通根。
- **等分配定理**: 二次形式の自由度 1 つあたり平均エネルギー $\frac12 k_BT$。
  温度の力学的定義: $\langle\frac12 m v_x^2\rangle = \frac12 k_BT$ — 分子デモの「温度計」の実装式。
- **エントロピー**: $S = k_B\ln W$(状態数)。混合・拡散の不可逆性の定量化。
- **大数の法則と中心極限定理**: マクロ量のゆらぎは $O(1/\sqrt{N})$ —
  「なぜマクロ世界は決定論的に見えるか」の説明。$N$ を UI で変えてゆらぎの縮小を見せる。

## 3. エンジン内での 3 つの役割

1. **輸送係数・状態方程式の由来**(ドキュメント+検証デモ): 粘性 $\mu$、熱伝導率 $k$、
   拡散係数 $D$ が分子運動から導出されること([02-kinetic-gas.md](02-kinetic-gas.md) §2.3)。
   連続体ドメインは実測値を使うが、分子デモで**同じ値がオーダーで再現される**ことを見せる。
2. **ゆらぎの注入**(実行時結合): ブラウン運動・熱雑音([03-diffusion-brownian.md](03-diffusion-brownian.md))。
   マクロでは平均化されて消える力を、微小スケールでは揺動散逸定理に従って明示的に加える。
3. **確率的模型の直接シミュレーション**(専用シーン): 気体分子動力学([02](02-kinetic-gas.md))、
   イジング模型([04-monte-carlo.md](04-monte-carlo.md))。

## 4. 粗視化の方法論(設計指針)

どの変数を残しどれを消すか(= どのドメインに委ねるか)の判定手順:

1. 対象の緩和時間 $\tau_{micro}$ と観測時間 $\tau_{obs}$ を比べる。
   $\tau_{micro} \ll \tau_{obs}$ なら平衡統計(平均値+ゆらぎ)で置換可能。
2. ゆらぎの相対振幅 $\sim 1/\sqrt{N}$ を見積もる。観測精度より小さければ決定論的マクロ変数のみ、
   大きければ確率過程(ランジュバン)として残す。
3. 保存量(エネルギー・運動量・粒子数)の輸送は必ずマクロ方程式に引き継ぐ
   (これが Navier-Stokes・熱伝導方程式の由来)。

具体例: 花粉粒(10 μm、水中)— 水分子の緩和は ps、観測は ms → 水は連続体+ランジュバンノイズ。
砂粒(1 mm)— ノイズ振幅が重力・摩擦の $10^{-9}$ 以下 → 純粋な剛体力学。

## 5. 適用スケールと限界

- 平衡・線形応答(揺動散逸)の範囲。強い非平衡(乱流の統計理論、ガラス転移)は対象外。
- 量子統計(ボース凝縮・フェルミ縮退)は説明ページのみ([14-quantum/03](../14-quantum/03-effective-models.md))。

## 7. 検証

- 分子デモの測定圧力・温度が状態方程式と一致([02](02-kinetic-gas.md) §7)。
- ゆらぎスケーリング: 区画内粒子数の分散 $\propto N$(相対 $1/\sqrt N$)。
- エントロピー: 自由膨張 $\Delta S = Nk_B\ln 2$(2 倍体積)を状態数カウントと熱力学式の両方で。

## 8. 実装フェーズ対応

Phase 4 で先行可(実装が軽い)。
