# 統計 02. 気体分子運動論 — 分子デモとマクスウェル=ボルツマン分布

crate: `sim-statistical`。気体を分子として直接シミュレーションする教育・検証シーン。

## 1. 担う現実の現象

気体の圧力・温度の分子的正体、断熱圧縮で熱くなる理由、拡散・混合、ブラウン運動の源。
遊び方の例: ピストンを押すと分子が速くなる(温度上昇)のを見る、速度ヒストグラムが
マクスウェル分布に収束する、軽い分子ほど速い(He vs N₂)。

## 2. 支配方程式

### 2.1 分子動力学(MD)

$N$ 個の粒子のニュートン力学。相互作用は 2 通り:

- **剛体球気体**(既定): 衝突は弾性(運動量・エネルギー保存の 2 体解)。
  希薄気体の理想モデル。イベント駆動でなく固定 $\Delta t$ + 重なり解消
  (決定論と World 統合のため)。
- **Lennard-Jones 気体**(オプション): $V(r) = 4\epsilon[(\sigma/r)^{12}-(\sigma/r)^6]$、
  velocity Verlet([01-math/03](../01-math/03-integrators.md) §2.3)。実在気体・凝縮(液滴生成)が見える。

### 2.2 導かれるマクロ法則(検証対象)

- **マクスウェル=ボルツマン速度分布**:
  $f(v) = 4\pi n \left(\frac{m}{2\pi k_BT}\right)^{3/2} v^2 e^{-mv^2/(2k_BT)}$
- **状態方程式**: 壁への運動量流束から $pV = Nk_BT$(導出: 1 分子の壁衝突の運動量 $2mv_x$ ×
  頻度 $v_x/2L$ の平均 × N)。
- **輸送係数**(剛体球、Chapman-Enskog 一次): $\mu = \frac{5}{16d^2}\sqrt{\frac{mk_BT}{\pi}}$ —
  空気の実測 $\mu = 1.8\times10^{-5}$ Pa·s が分子直径 $d \approx 3.7\times10^{-10}$ m から
  再現されることを見せる([01](01-micro-macro-bridge.md) §3 の役割 1)。

## 3. 状態表現

```rust
pub struct GasSim {
    pub particles: ParticleSet,          // [01-math/02] §6
    pub interaction: Interaction,        // HardSphere{d} / LennardJones{eps, sigma}
    pub container: Container,            // Box{walls} / Piston{wall_id, body: BodyId}
    pub hash: SpatialHash,
    pub wall_momentum_accum: Vec<f64>,   // 圧力測定 (壁ごと)
}
```

- スケーリング: 実気体の $N \sim 10^{25}$ は不可能。**代表粒子** $10^3$–$10^4$ 個で
  統計法則を見せる(ゆらぎが実際より大きいことは $1/\sqrt N$ 表示で明示 —
  これ自体が教材)。物理単位は実分子(質量・直径・温度は本物)、密度だけデモ用に希薄化。

## 4. 数値解法

```text
gas_step(dt):                        # dt: 平均自由時間の 1/10 以下に sub-step
  hash.rebuild
  衝突検出 (剛体球): 重なりペア → 弾性衝突解 (衝突法線方向の速度交換)
  壁衝突: 鏡面反射 (断熱壁) or 熱壁 (§4.2)、運動量を wall_momentum_accum へ
  ピストン壁: 壁速度を考慮した反射 (動く壁 → 分子が加減速 = 断熱圧縮の仕事)
  LJ の場合: 力計算 (カットオフ 2.5σ) → velocity Verlet
```

- **熱壁(サーモスタット)**: 壁温度 $T_w$ の壁に当たった分子を
  マクスウェル流束分布(壁法線成分は Rayleigh 分布)で再放出。物理的に正しい熱浴。
  大域サーモスタット(Berendsen 等)は非物理的なので使わない(検証ツールとして)。
- 圧力測定: $p = \frac{\sum \Delta p_{wall}}{A\,\Delta t_{avg}}$(移動平均窓 ~1000 step)。
- 温度測定: $T = \frac{m}{3k_B N}\sum v_i^2$(重心運動を除く)。

## 5. 適用スケールと限界

- 代表粒子数の制約(§3): 定量再現は分布形・状態方程式・スケーリング則。
  実密度の平均自由行程などは解析式で補完表示。
- 剛体球は単原子的(回転自由度なし)— 二原子分子の $c_v = \frac52 R$ は
  回転エネルギーを持つ剛体ダンベル(オプション、Phase 5+)か説明ページで。
- 化学反応・電離はしない。

## 6. 他ドメインとの結合

- 力学: ピストン(剛体)との運動量交換 — **分子から熱力学へ**の実演
  ([12-thermal/01](../12-thermal/01-thermodynamics-laws.md) §4.3 の GasCompartment と
  同一設定を並べ、集中定数モデルとの一致を見せる統合デモ)。
- 熱: 熱壁の温度を熱ドメインのノードと接続(分子が壁を暖める、Phase 5)。

## 7. 検証

- MB 分布への収束: 初期単一速度 → ヒストグラムの $\chi^2$ が MB と適合(数百衝突/粒子後)。
- $pV = Nk_BT$: 体積・温度を変えて ± 2%($N = 10^4$、ゆらぎ込み)。
- 断熱圧縮: 準静的ピストンで $TV^{\gamma-1}$ 一定($\gamma = 5/3$ 単原子)± 2%。
- エネルギー保存(断熱壁・剛体球): 機械精度。運動量保存(周期境界): 同。
- 等分配: $\langle v_x^2\rangle = \langle v_y^2\rangle = \langle v_z^2\rangle$($1/\sqrt N$ 内)。
- Chapman-Enskog: 粘性のオーダー再現(±30%、小 N の限界を明記)。

## 10. 性能プロファイル

- ホットスポット: 分子衝突検出、壁衝突、LJ 力計算。
- 目標アルゴリズムとオーダー: 空間ハッシュ + Morton 順で $O(N)$。velocity Verlet。
- SoA レイアウト: position/velocity/mass 別配列。定期 Morton ソート。
- 並列化単位: 粒子を近傍セルで rayon 分割。
- SIMD 対象カーネル: 衝突判定・LJ 力のバッチ。
- GPU 適性: 高(多数粒子)。
- ベンチ: 10⁴ 粒子(予算 2 ms、S1-S3)。
