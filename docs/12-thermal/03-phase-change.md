# 熱 03. 相変化 — 融解・凝固・蒸発・潜熱

crate: `sim-thermal`。

## 1. 担う現実の現象

氷が融ける、水が凍る・沸く、汗の蒸発冷却、鍋の湯が沸騰しても 100 °C のまま。
遊び方の例: 氷入りの飲み物の温度が 0 °C 付近に張り付く様子、標高(気圧)と沸点の関係、
氷が融けても水位が変わらない(アルキメデスとの統合)。

## 2. 支配方程式

### 2.1 潜熱と相転移条件

一次相転移では、転移温度で熱が温度上昇でなく相変化に使われる:

$$Q = m\,L \qquad (L: 潜熱\ \mathrm{[J/kg]})$$

水: 融解熱 $L_f = 334$ kJ/kg(0 °C)、蒸発熱 $L_v = 2257$ kJ/kg(100 °C, 1 atm)。

### 2.2 エンタルピー法(採用)

温度でなく**比エンタルピー** $H$ [J/kg] を状態変数とし、$T(H)$ を区分関数で引く:

$$T(H) = \begin{cases}
T_m + (H - H_{sol})/c_{p,s} & H < H_{sol} \quad(\text{固相})\\
T_m & H_{sol} \le H \le H_{sol} + L_f \quad(\text{混合相})\\
T_m + (H - H_{sol} - L_f)/c_{p,l} & H > H_{sol} + L_f \quad(\text{液相})
\end{cases}$$

利点: 相境界の追跡(Stefan 問題の界面条件)を陽に解かず、熱収支だけで融解・凝固が正しく進む。
混合相の**液相率** $\phi = (H - H_{sol})/L_f \in [0,1]$ が「半分融けた氷」を表す。

### 2.3 蒸発(表面律速モデル)

沸騰前の蒸発は表面からの物質移動:

$$\dot m_{evap} = h_m A\,(\rho_{v,sat}(T_s) - \rho_{v,\infty})$$

$h_m$: 物質移動係数(熱対流係数から Lewis 関係 $h_m \approx h/(\rho c_p Le^{2/3})$ で推定)、
$\rho_{v,sat}$: 飽和蒸気密度(Antoine 式から $p_{sat}$、理想気体で密度化)。
Antoine 式(水、mmHg・°C、範囲 1–100 °C): $\log_{10} p_{sat} = 8.07131 - \frac{1730.63}{233.426 + T}$。
蒸発は $\dot Q = L_v \dot m$ の冷却として表面ノードに作用(汗・打ち水)。
沸騰(核沸騰の熱流束)は経験式(Rohsenow)を Phase 5 検討とし、当面「$T = T_{boil}$ に
クランプ + 入熱を全て蒸発に回す」単純モデル。

## 3. 状態表現

```rust
pub struct PhaseState {
    pub enthalpy: f64,        // 比エンタルピー [J/kg] (ThermalNode の T はこれから導出)
    pub phase: Phase,         // Solid / Mixed{liquid_fraction} / Liquid / (Gas)
    pub mass: f64,            // 蒸発で減少
}
```

- 固→液で剛体としての振る舞いが変わる(氷塊→水): Phase 3 では「液相率 100% で
  剛体を消して SPH 粒子 or 水域体積に振り替える」イベント駆動の変換で表す。
  部分融解の力学(軟化)は対象外(§5)。

## 4. 数値解法

熱ステップの変更は最小: 熱流の積算先を $T$ でなく $H$ にし、$T(H)$ で温度を引く。

```text
thermal_step_with_phase(dt):
  solve linear network for T (混合相ノードは T = T_m の Dirichlet 扱い)
  for each node: H += (Q̇_net * dt) / mass
  T = T_of_H(H);  phase = phase_of_H(H)
  蒸発ノード: ṁ = evap_rate(T_surf, env);  mass -= ṁ dt;  H -= L_v ṁ dt / mass
  相変化完了イベント (液相率 0↔1 通過) を EventQueue へ
```

- 混合相を Dirichlet 扱いにすることで陰解法の行列は線形のまま(標準的なエンタルピー法の
  実装手筋。Voller & Cross 1981)。
- 質量変化は浮力・慣性に反映(氷が軽くなる)— 剛体の質量・慣性テンソル再計算をイベントで。

## 5. 適用スケールと限界

- 相変化は材料表に $T_m, L_f, T_{boil}, L_v$ がある純物質のみ。合金・混合物の凝固域、
  過冷却・核生成は対象外(過冷却は将来、確率イベントとして統計ドメインと接続可能)。
- 「半融けの固体の力学」(軟化・崩壊)はモデル化しない。液相率 1 で離散的に変換する。
- 昇華・凝縮(結露)は Phase 5 検討。

## 6. 他ドメインとの結合

- 流体: 融けた水の生成(SPH 粒子 spawn / 水域体積追加)、蒸発質量→湿度(将来)。
- 力学: 質量・形状変化イベント。
- 気圧: GasCompartment の圧力で $T_{boil}$ を Antoine 逆算(山の上の沸点デモ)。

## 7. 検証

- 氷融解時間: 0 °C の氷球(半径 r)を $T_\infty$ の水中に置いた準定常解
  $t_{melt} = \rho_i L_f r^2 / (2 k_w (T_\infty - T_m))$ 程度(オーダー一致 ±20%、
  対流依存を明記)。
- 温度プラトー: 加熱曲線が $T_m$ で水平になり、プラトー長 = $mL_f/\dot Q \pm 1\%$。
- エネルギー: 相変化を含む全過程で台帳 residual < 10⁻³。
- 沸点の気圧依存: 0.7 atm で ≈ 90 °C(Antoine 式 ± 1 °C)。

## 8. 実装フェーズ対応

Phase 3: エンタルピー法(融解・凝固)+ プラトーデモ。Phase 4: 蒸発冷却・沸点気圧依存。
Phase 5: 沸騰熱流束・凝縮・過冷却。

## 10. 性能プロファイル

- ホットスポット: エンタルピー法の温度導出・相判定(熱ステップに相乗り)。
- 目標アルゴリズムとオーダー: $T(H)$ 区分関数は $O(N)$。混合相 Dirichlet で行列は線形維持。
- SoA レイアウト: enthalpy/phase/mass をノード配列に追加列。
- 並列化単位: 熱ステップと同じ(ノード/格子並列)。
- SIMD 対象カーネル: $T(H)$・液相率のバッチ評価。
- GPU 適性: 中(格子相変化は GPU 向き)。
- ベンチ: 氷融解・温度プラトー(T7)。
