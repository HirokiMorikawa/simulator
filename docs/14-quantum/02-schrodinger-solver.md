# 量子 02. シュレディンガーソルバ — split-step Fourier 法とデモ設計

crate: `sim-quantum`。1 粒子 TDSE の数値解(顕微鏡モード専用シーン)。

## 1. 担う現実の現象(デモとして)

- **二重スリット**: 電子 1 個ずつの検出点が積み上がって干渉縞になる(波動性と粒子性)。
- **トンネル効果**: 古典では越えられない障壁を波束が確率的に透過。
- **束縛状態**: 井戸型ポテンシャルの離散準位、波動関数の節。
- **波束の広がり**: 自由粒子の分散(不確定性関係の時間発展)。
- **調和振動子**: コヒーレント状態が古典振動を再現(対応原理の実演)。

## 2. 支配方程式

TDSE([01](01-role-and-limits.md) §2)。内部は原子単位($\hbar = m_e = 1$):

$$i\frac{\partial\psi}{\partial t} = \left[-\frac{1}{2}\nabla^2 + V(\mathbf{x},t)\right]\psi$$

## 4. 数値解法 — split-step Fourier(Strang 分割)

ハミルトニアンを $\hat H = \hat T + \hat V$ に分け、時間発展演算子を分割:

$$\psi(t+\Delta t) = e^{-i\hat V\Delta t/2}\; \mathcal{F}^{-1}\, e^{-i k^2\Delta t/2}\, \mathcal{F}\; e^{-i\hat V\Delta t/2}\,\psi(t) + O(\Delta t^3)$$

- $\hat V$ は実空間で対角(位相回転)、$\hat T$ は波数空間で対角 — FFT で往復するだけ。
- **ユニタリ性が厳密**(各因子が位相回転)→ ノルム $\int|\psi|^2 dx = 1$ が機械精度で保存。
  これが検証の錨になる。
- 二次精度(Strang)。$\Delta t$ の制約は精度のみ(位相エイリアス回避に
  $\Delta t < \pi/\max(k^2/2, V)$)。

### 4.2 初期状態

ガウス波束: $\psi_0 \propto \exp\left[-\frac{(x-x_0)^2}{4\sigma^2} + ik_0x\right]$
(運動量 $\hbar k_0$、幅 $\sigma$)。固有状態は虚時間発展($t \to -i\tau$ で基底状態に収束、
直交化を挟んで励起状態)で用意する。

### 4.3 境界

周期境界(FFT の性質)を吸収マスク(境界近傍で $\psi \mathrel{*}= \cos^{1/8}$ 型の減衰)で
擬似開放境界化。吸収された確率は「検出されず飛び去った」として集計(ノルム台帳)。

### 4.4 測定(検出スクリーン)

スクリーン位置の $|\psi|^2$ を確率分布として、決定論的 PRNG(STREAM_QUANTUM、
[01-math/04](../01-math/04-random.md))で検出点をサンプル。
1 電子=1 サンプル、波動関数はリセットして次の電子 — 干渉縞が点描で積み上がる。

## 5. 適用スケールと限界

- 1 粒子・非相対論・スピンなし。ポテンシャルは実数(吸収は現象論マスク)。
- 2D は 512² まで(FFT 予算)。3D は非対応(メモリ)。
- 時間結合なし(専用シーン、[01](01-role-and-limits.md) §4)。

## 6. 他ドメインとの結合

- FDTD の二重スリット([13-electromagnetism/03](../13-electromagnetism/03-maxwell-fdtd.md) §7)と
  並置デモ: 「電子の波」と「光の波」の干渉が同じ数学であることを見せる。
- 統計: 多数回測定のヒストグラム → ボルン則の頻度解釈(大数の法則のデモ、
  [15-statistical/01](../15-statistical/01-micro-macro-bridge.md))。

## 7. 検証

- ノルム保存: $|1 - \int|\psi|^2| < 10^{-12}$(吸収なし設定)。
- 自由波束の広がり: $\sigma(t) = \sigma_0\sqrt{1 + (\hbar t/(2m\sigma_0^2))^2}$ ± 0.1%。
- 無限井戸固有値: $E_n = \frac{n^2\pi^2\hbar^2}{2mL^2}$、虚時間発展の結果が n=1..5 で ± 0.1%。
- 調和振動子: $E_n = \hbar\omega(n+\frac12)$、コヒーレント状態の $\langle x\rangle(t)$ が
  古典解と一致(± 0.1%、エーレンフェスト)。
- トンネル透過率: 矩形障壁の解析式
  $T = \left[1 + \frac{V_0^2\sinh^2(\kappa a)}{4E(V_0-E)}\right]^{-1}$ と
  波束シミュレーションの透過確率が一致(± 2%、波束のエネルギー幅を考慮した重み付き平均で)。
- 二重スリット: 縞間隔 $\Delta y = \lambda_{dB} D/d$($\lambda_{dB} = h/p$)± 2%。
- 収束次数: $\Delta t$ 半減で誤差 1/4(二次)。
