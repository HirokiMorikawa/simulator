# 量子 03. 有効モデル — 物性・半導体・光・化学への橋渡し

crate: `sim-quantum`(式・データ)+ docs(説明)。量子力学がマクロドメインに供給する
法則とパラメータのカタログ。各項目は「マクロ側の式・値」「量子的由来(要約)」「エンジンでの実装先」を持つ。

## 1. 担う現実の役割

「なぜ物はすり抜けないのか」「なぜ銅は通しガラスは通さないのか」「なぜ熱した鉄は赤いのか」—
日常物理の**根拠**を提供し、UI の「なぜ?」リンクから読める形にする。
値そのものは実測(MaterialDb)を使い、第一原理計算はしない([01](01-role-and-limits.md) §5)。

## 2. カタログ

### 2.1 固体の硬さ・剛性(→ 力学)

- マクロ側: ヤング率 E、接触の非貫入(衝突検出の前提)。
- 由来: パウリの排他原理 + 電子雲の重なりの反発。結合エネルギー曲線(Lennard-Jones 型
  $V(r) = 4\epsilon[(\sigma/r)^{12} - (\sigma/r)^6]$)の曲率が弾性率になる:
  $E \sim V''(r_0)/r_0$。オーダー確認: $\epsilon \sim 1$ eV, $r_0 \sim 3\times10^{-10}$ m →
  $E \sim 10^{10}$–$10^{11}$ Pa(実測の金属・セラミックスと一致)。
- 実装: MaterialDb の E。LJ ポテンシャルは統計ドメインの分子デモ([15-statistical/02](../15-statistical/02-kinetic-gas.md))で共用。

### 2.2 導電性とバンド構造(→ 回路)

- マクロ側: 抵抗率 ρ_e(MaterialDb)、オームの法則。
- 由来: 周期ポテンシャル中の電子はバンドを作る(1D Kronig-Penney モデルで説明可能 —
  シュレディンガーソルバの周期ポテンシャルデモと接続)。部分充填バンド=金属、
  ギャップ $E_g \gg k_BT$=絶縁体、$E_g \sim 1$ eV=半導体。
- ドルーデ模型: $\rho_e = m_e/(n e^2 \tau_c)$ — 銅($n = 8.5\times10^{28}$ /m³)で
  $\tau_c \sim 2.5\times10^{-14}$ s を逆算(妥当な散乱時間)。

### 2.3 ダイオード・太陽電池(→ 回路)

- マクロ側: Shockley 式 $i = I_s(e^{v/nV_T} - 1)$([13-electromagnetism/02](../13-electromagnetism/02-circuits.md))。
- 由来: pn 接合のキャリア統計(ボルツマン因子 $e^{qv/k_BT}$)。$V_T = k_BT/q$ が
  「量子統計の温度」を回路に持ち込む。
- LED の色: $\lambda = hc/E_g$(GaN 青 ≈ 2.8 eV → 443 nm)。光電効果 $E = h\nu - W$ は
  光学デモ(Phase 5)。

### 2.4 黒体放射・原子スペクトル(→ 光学・熱)

- マクロ側: プランク則・SB 則([13-electromagnetism/04](../13-electromagnetism/04-light-optics.md) §2.3)。
- 由来: 電磁場モードのエネルギー量子化 $E = n h\nu$(古典等分配は紫外破綻 —
  レイリー・ジーンズとの比較曲線を「なぜ?」ページに載せる)。
- 原子輝線: 水素バルマー系列 $\frac{1}{\lambda} = R_H(\frac{1}{4} - \frac{1}{n^2})$
  ($R_H = 1.097\times10^7$ /m)— ネオンサイン・炎色反応の光源スペクトルとして光学ドメインに供給。

### 2.5 化学反応速度(→ 熱)

- マクロ側: アレニウス式 $k_{rate} = A e^{-E_a/(RT)}$ — 熱ドメインの発熱反応スクリプト
  (燃焼の簡易モデル: 燃料量 × k_rate × 発熱量)。
- 由来: 活性化障壁の熱的乗り越え(ボルツマン因子)+ トンネル補正(低温で速い —
  トンネルデモとの接続)。
- 例: 「温度 10 K 上昇で反応速度 2 倍」($E_a \approx 50$ kJ/mol 付近の経験則)を
  デモで確認できる。

### 2.6 比熱の量子起源(→ 熱)

- マクロ側: $c_p$(MaterialDb)。
- 由来: デュロン=プティ則 $c_v = 3R/M$(古典等分配)が高温極限。
  低温では格子振動の量子化(アインシュタイン/デバイモデル)で減少 —
  「なぜ金属の比熱はだいたい 3R/M か」の説明と、実測値との比較表。

## 3〜6. 状態・解法・限界・結合(要約)

- 実装は「式 + 定数」がほとんど(回路素子・光源スペクトル・反応速度関数)。
  専用の時間発展ソルバは持たない。
- 限界: すべて平衡・準平衡の有効式。非平衡量子動力学は対象外。
- 結合: 供給先は各節に明記(回路・光学・熱・材料DB)。

## 7. 検証

- 各式の引用値再現(バルマー線波長、$V_T$ = 25.85 mV @300K、デュロン=プティとの比較)を
  ユニットテスト化。
- 統合: LED デモ(回路の順方向電圧 ≈ $E_g/e$ と発光色の整合)。
