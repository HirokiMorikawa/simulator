# 電磁気 04. 光と光学 — 幾何光学・反射屈折・放射

crate: `sim-em`。光の物理(電磁波の短波長極限)。描画レンダラとは役割が異なる(§5)。

## 1. 担う現実の現象

鏡の反射、水中の物のずれて見える位置、レンズの結像、プリズムの分光、虹、
全反射(光ファイバ)、熱放射の色(赤熱→白熱)。
遊び方の例: レンズの焦点距離を測る、臨界角を探す、プリズムで白色光を分ける、
水槽の中の棒の見かけの折れ曲がり角の予測と確認。

## 2. 支配方程式

### 2.1 幾何光学(アイコナール近似)

波長 ≪ 幾何形状のとき、マクスウェル方程式は光線方程式に退化する:
一様媒質では直進、界面で:

- **反射**: $\theta_r = \theta_i$(法線に対し同角、同一平面)
- **屈折(スネル則)**: $n_1\sin\theta_1 = n_2\sin\theta_2$
- **全反射**: $\sin\theta_c = n_2/n_1$($n_1 > n_2$)
- **フレネル係数**(強度分配、非偏光平均):
  $R = \frac{1}{2}(R_s + R_p)$、$R_s = \left|\frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2}\right|^2$ 等。透過 $T = 1 - R$。

### 2.2 分散(波長依存の屈折率)

Cauchy 式 $n(\lambda) = A + B/\lambda^2$(可視域で十分)。
BK7 ガラス: $A = 1.5046$, $B = 4.2\times10^{-15}$ m²(Sellmeier からの近似)。プリズム・虹の源。

### 2.3 熱放射(プランク則)

温度 $T$ の物体の分光放射輝度:

$$B_\lambda(T) = \frac{2hc^2}{\lambda^5}\frac{1}{e^{hc/(\lambda k_B T)}-1}$$

ウィーンの変位則 $\lambda_{max} T = 2.898\times10^{-3}$ m·K。
全放射はシュテファン=ボルツマン([12-thermal/02](../12-thermal/02-heat-transfer.md) §2.3)と整合。
量子ドメインがこの式の由来(エネルギー量子化)を説明する([14-quantum/01](../14-quantum/01-role-and-limits.md))。

## 4. 数値解法

- 光線と形状の交差は衝突検出の ray-cast を再利用([10-mechanics/02](../10-mechanics/02-collision-detection.md))。
- 分光は波長サンプリング(可視 380–780 nm を 16–64 サンプル、既定 16)→ CIE 等色関数で
  RGB へ(表示層)。
- 結像・強度分布は光線密度の集計(スクリーン格子へのビニング)。
  回折限界以下の解像は主張しない(§5)。
- 決定論: 光線分岐は決定的(確率的パストレーシングはしない。分岐爆発は深さ・パワー打切りで制御)。

## 5. 適用スケールと限界

- 幾何光学の成立: 開口・構造 ≫ 波長。回折・干渉は扱わない(それは FDTD
  [03](03-maxwell-fdtd.md) と量子 [14-quantum/02](../14-quantum/02-schrodinger-solver.md) の領分 —
  三者の分業を UI で案内)。薄膜干渉・ホログラフィは対象外。
- 偏光はフレネル平均のみ(偏光追跡は Phase 5+)。
- **描画レンダラとの関係**: Three.js の見た目(ラスタライズ+経験的シェーディング)は物理ではない。
  光学シミュレーションの結果(光線経路・スクリーン像・スペクトル)は**物理観測データ**として
  オーバーレイ表示する。「見た目」と「物理の光」を混同させない設計。

## 6. 他ドメインとの結合

- 熱: 放射スペクトル(赤熱の色)を温度から生成。光吸収 → 加熱(黒い物は日向で熱い、Phase 5)。
- 量子: プランク則・原子スペクトル線([14-quantum/03](../14-quantum/03-effective-models.md))を光源スペクトルに。
- 力学: 幾何(鏡・レンズ・水面)の位置姿勢を共有。水面([11-fluid](../11-fluid/))の法線 → 屈折のゆらぎ。

## 7. 検証

- スネル則: 数値屈折角 = 解析、機械精度。臨界角の全反射切替の正確さ。
- 薄レンズ: 球面レンズの結像 $1/f = (n-1)(1/R_1 - 1/R_2)$、近軸光線で焦点距離誤差 < 1%、
  非近軸で球面収差が**正しく出る**こと(近軸公式からの系統的ズレ)。
- フレネル: 垂直入射の反射率 $R = ((n_1-n_2)/(n_1+n_2))^2$(ガラス 4%)、
  ブリュースター角で $R_p = 0$。
- プリズム: 頂角 60° BK7 の最小偏角 → $n$ 逆算誤差 < 0.5%。
- エネルギー: 各界面で $R + T = 1$、系全体で入射パワー = 吸収+射出(< 10⁻⁹)。
- プランク則: ピーク波長のウィーン則一致、全積分の SB 則一致(数値積分 < 0.1%)。

## 10. 性能プロファイル

- ホットスポット: 光線と形状の交差、分岐トレース。
- 目標アルゴリズムとオーダー: BVH で $O(\log n)$ 交差。波長サンプリング。
- SoA レイアウト: 光線バッチ(origin/dir/power/wavelength)。
- 並列化単位: 光線を rayon 分割(光線独立)。
- SIMD 対象カーネル: レイ-形状交差、フレネル・スネル評価。
- GPU 適性: 高(レイトレースは GPU 向き。[17-rendering](../17-rendering/) と共通基盤)。
- ベンチ: レンズ・プリズム・水中(E9-E12)。
