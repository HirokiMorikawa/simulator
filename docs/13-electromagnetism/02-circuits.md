# 電磁気 02. 回路 — 集中定数・MNA 法・電池とモーター

crate: `sim-em`。電気を「遊べるインフラ」にする中核。乗り物・ロボット(エンティティ層)の動力の電気側。

## 1. 担う現実の現象

電池と豆電球、スイッチ、コンデンサの充放電、コイルの遅れ、モーターを回す、発電機、
ヒューズが切れる、電熱線が赤くなる。
遊び方の例: RC 回路の時定数測定、直並列の明るさ比べ、モーターに負荷をかけると電流が増える。

## 2. 支配方程式

集中定数近似($L_{回路} \ll \lambda = c/f$、[00-foundation/02](../00-foundation/02-scale-ladder.md) §3):
キルヒホッフ電流則(KCL)$\sum_k i_k = 0$(各ノード)と素子の構成則。

| 素子 | 構成則 |
|---|---|
| 抵抗 | $v = Ri$ |
| コンデンサ | $i = C\,dv/dt$ |
| インダクタ | $v = L\,di/dt$ |
| 電圧源(電池) | $v = V_{emf} - R_{int}\,i$、容量 [Ah] を持ち枯渇 |
| ダイオード | Shockley $i = I_s(e^{v/nV_T}-1)$([14-quantum/03](../14-quantum/03-effective-models.md) が根拠を説明) |
| DCモーター | $v = Ri + L\,di/dt + k_e\omega$(逆起電力)/ $\tau = k_t i$([05](05-em-mechanics-coupling.md)) |

## 4. 数値解法

- **時間離散化**: 各動的素子を「コンパニオンモデル」(等価抵抗+電流源)に変換して代数化する
  SPICE 標準手法。後退 Euler(既定、L安定・単調)/ 台形則(高精度モード、二次)。
  例(後退 Euler): コンデンサ → $G_{eq} = C/\Delta t$, $I_{eq} = (C/\Delta t)v^n$。
- **非線形素子**(ダイオード・モーター飽和): Newton-Raphson 反復(各ステップ 2–5 回)。
  収束保護: 電圧ステップ制限(ダイオードの指数発散防止、SPICE の限流手法)。
- **線形ソルバ**: 回路規模は小さい(<10³ 節点)ので密 LU で十分。トポロジ固定なら
  スパースパターン再利用。
- **タイムステップ**: 回路の時定数($RC$, $L/R$)は world $\Delta t$(8.3 ms)より速いことが多い
  → 回路は sub-stepping(既定 $\Delta t_{circ} = \Delta t/32 \approx 0.26$ ms、
  時定数検査で自動増減)。無条件安定なので精度要件のみで決める。
- スイッチ開閉はイベント(ステップ境界に整列、決定論)。

## 5. 適用スケールと限界

- 集中定数のみ: 伝送線路効果・アンテナ・EMI は対象外(波動は [03-maxwell-fdtd.md](03-maxwell-fdtd.md) の
  専用シーン)。$f < 1$ MHz・回路サイズ < 10 m なら妥当($\lambda > 300$ m)。
- 電池は理想 EMF + 内部抵抗 + 容量。電気化学(放電曲線の非線形・温度依存)は
  簡易テーブル補正まで(Phase 5)。
- 半導体はダイオードのみ(トランジスタは Phase 5+、[14-quantum/03](../14-quantum/03-effective-models.md) の
  拡張として)。

## 6. 他ドメインとの結合

- **熱**: 各抵抗素子のジュール熱 $P = i^2R$ を素子に紐づく ThermalNode へ
  ([12-thermal/02](../12-thermal/02-heat-transfer.md) §4.4)。温度依存抵抗
  $R(T) = R_0(1 + \alpha_R(T-T_0))$(銅 $\alpha_R = 3.9\times10^{-3}$/K)で逆結合 —
  電熱線・ヒューズ(溶断 = 相変化イベント)のデモ。
- **力学**: モーター・発電機([05](05-em-mechanics-coupling.md))、スイッチの機械操作。
- **台帳**: 電池化学エネルギー → 電場/磁場エネルギー($\frac12CV^2, \frac12LI^2$)→ ジュール熱/機械仕事。

## 7. 検証

- RC 過渡: $v(t) = V(1-e^{-t/RC})$、$\tau$ 誤差 < 0.5%(台形則)/ 一次収束(後退 Euler)。
- RLC 共振: 減衰振動の周波数 $\omega = \sqrt{1/LC - (R/2L)^2}$ ± 1%。
- 分圧・分流: 直並列の解析値と機械精度一致。
- ダイオード整流: 半波整流の平均電圧(± 2%)。
- エネルギー: 電池供出 = ジュール熱 + 蓄積 + 機械仕事(residual < 10⁻⁶ 相対)。
